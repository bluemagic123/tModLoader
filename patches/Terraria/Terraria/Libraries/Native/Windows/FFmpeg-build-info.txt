These FFmpeg libraries contain _only_ the wmav2 decoder
I built FFmpeg using https://github.com/m-ab-s/media-autobuild_suite
Say no to basically all the configuration options
Then edit build/ffmpeg_options.txt
Replace all the switches with:

--disable-autodetect
--disable-debug
--disable-programs
--disable-doc
--disable-avdevice
--disable-avformat
--disable-pixelutils
--disable-network
--disable-everything
--enable-decoder=wmav2
